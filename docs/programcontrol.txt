Program control: connecting the command line to the file handler (request file contents and return readable text)
- no args: provide list of files available (team member B) and corresponding number
    ex: input ->  java topsecret
        output -> 01 filea.txt
                  02 fileb.txt
                  03 filec.txt
- one arg: display content of file
    ex: input ->  java topsecret 01
        output -> (content of filea.txt)
- two args: specifies an alternate key to use when deciphering (none = just use default key)

flow of the program:
TopSecret.main()
- create ProgramControl
- create UserInterface with ProgramControl
- UserInterface.run(args)
    - calls ProgramControl.getFileList() or
    - calls ProgramControl.getFileContent(fileNumber, keyPath)
        - calls FileHandler.listFiles() or FileHandler.getFile()
        - returns results to UserInterface
- UserInterface handles CipherDecrypter
- UserInterface displays results to user
